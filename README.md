### 要求
- Discord Server
- node
- JavaScript
- 腦袋

###  主要特點
- JavaScript V14
- Slash Commands “/” 
- 加入日誌 (當我玩家進到伺服器時)
- 離開日誌 (當玩家與離開伺服器)
- 申請表單
- 領取身分組
- Ticket (有紀錄)
